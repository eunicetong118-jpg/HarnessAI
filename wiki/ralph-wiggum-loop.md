# Ralph Wiggum Loop

A pattern for continuous agent iteration.

## Concept
The agent works in a loop (often via hooks or scripts) to continuously refine and iterate until a goal is met or a reviewer is satisfied.

## Application
- Used to drive PRs to completion.
- Agent reviews its own changes locally.
- Requests external agent reviews.
- Responds to feedback in a loop.
- "I'm in a loop!" - inspired by the character's simplicity and persistence.
