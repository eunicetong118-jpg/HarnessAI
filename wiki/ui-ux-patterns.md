# UI & UX Patterns

Recurring solutions for common user interface and user experience challenges.

## Layout & Responsive Patterns
- **Fluid Layouts**: Proportional widths rather than fixed pixels.
- **Priority Columns**: In responsive tables, hide less important columns on small screens.
- **Horizontal Overflow**: Allow tables/content to scroll horizontally on small screens.
- **Layout Flip**: Reorienting headers or navigation based on viewport.

## Responsive Media
- **Resolution Switching**: Using `srcset` to serve different images based on device pixel density.
- **Art Direction**: Using the `<picture>` element to serve differently cropped or styled images for different viewports.

## Product & Validation Patterns
- **Problem-Market Fit**: Validating if the problem is worth solving before building.
- **Friction Reduction**: Minimizing the effort required for users to complete tasks.
- **Design Systems**: Centralized standards to prevent interface inconsistency and slow development.
- **Persuasive Patterns**: Techniques to influence user behavior (e.g., social proof, scarcity).
