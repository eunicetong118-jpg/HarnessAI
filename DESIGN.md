# Design System — Rebatengine

## Product Context
- **What this is:** High-performance brokerage rebate portal.
- **Who it's for:** Real traders looking to save on trading costs.
- **Project type:** Web app / Dashboard / Admin portal.

## Aesthetic Direction
- **Direction:** Dark Modern Fintech
- **Decoration level:** Intentional (subtle gradients, translucent borders)
- **Mood:** Professional, high-tech, and savings-focused.

## Typography
- **Display/Hero:** Inter — Bold, leading-tight.
- **Body:** Inter — Clean, readable.
- **Data/Tables:** Inter — Using tabular numbers where appropriate.
- **Scale:** Modular scale for hierarchy.

## Color
- **Background:** `#0f0b1f` (design-bg)
- **Surface:** `#15102b` (design-surface)
- **Card:** `#1e183d` (design-card)
- **Primary Accent:** `#ec4899` (design-pink)
- **Secondary Accent:** `#9333ea` (design-purple)
- **Neutrals:** Grays ranging from `#9ca3af` to white/translucent white.

## Spacing
- **Base unit:** 4px
- **Density:** Comfortable
- **Scale:** Standard Tailwind spacing scale.

## Layout
- **Approach:** Grid-disciplined
- **Border radius:** rounded-xl (12px) for cards and inputs.

## Motion
- **Approach:** Minimal-functional
- **Easing:** ease-in-out transitions.
