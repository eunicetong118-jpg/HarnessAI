<feature>
  <name>Rebate Processing Engine</name>
  <files>services/rebate.service.ts, __tests__/services/rebate.service.test.ts</files>
  <behavior>
    - Formula: volume * rebatePerLot * 0.80 (converted to BigInt cents).
    - Deduplication: Skip trades where tradeId exists in ProcessedTrade or Ledger referenceId.
    - Aggregation: Sum all rebates for a user in the batch and create ONE Ledger entry.
    - Tracking: Insert a record into ProcessedTrade for every successfully processed tradeId.
  </behavior>
  <implementation>
    1. Implement `calculateRebate(volume, rebatePerLot)` helper using BigInt.
    2. Implement `processBatch(trades)`:
       - Filter out trades already in ProcessedTrade.
       - Calculate individual rebates.
       - Group by userId and sum.
       - Use a Prisma transaction to:
         a. Insert ProcessedTrade records.
         b. Insert aggregated Ledger entries.
    3. Return a summary: { processed: N, skipped: N, totalCents: BigInt }.
  </implementation>
</feature>
